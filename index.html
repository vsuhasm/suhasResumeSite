My Projects

I worked with Prof. Peter Yoon from the Computer Science Department at Trinity on a research project titled "GPU-accelerated VLSI Routing using Group Steiner Trees"

I was responsible for writing code using parallel programming frameworks such as CUDA and MPI to accelerate an approximation algorithm on the Blue Waters Supercomputer . Click here to check out the source code for this project on github.

I built a web app named Muze at HackMIT 2015.

Muze is a location-based news website that takes in the user's geolocation and displays the top local news in the user's area.

I used pure front end and google's geolocation and news API to build this web app. Click here to check out the source code for this project on github.

As part of Mobile Computing class at Trinity College, I am building an android app named TrinTracker.

TrinTracker is an app that tracks the Trinity's on-campus shuttle's location and updates it on a map view for the user.

The project is still in it's intial stages Click here to check out more details about this project on my google sites portfolio.
